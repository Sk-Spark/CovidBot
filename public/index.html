We have created a Telegram Bot. Which can help you to keep an active monitoring on the number of open vaccination slots available in your area.
You can add this Bot to your Telegram chats and start monitoring open vaccination slots in your area after adding pin code.
You will get a push notification as soon as we find any open vaccination slots in your added pin codes.
How to add Vaccination Bot ?

  • In order to use Vaccination Bot, you must have a telegram account and telegram app install on your mobile or desktop.
    You can download & install Telegram App from below listed links: Download Telegram Apps.
  • After installing Telegram create an account on it.
  • After creating account click on below link to add Vaccination Bot to your chat. Vaccination bot Link
After adding Vaccination Bot, use provided command to add pin code in order to start monitoring for available vaccination slots.
You can use ‘/help’ command to list available commands.
  We suggest you to add 2 - 3 pin codes for a better chance of getting a slot as soon as possible.
 In Vaccination Bot you can also set ages for which you want to monitor vaccination slots.
